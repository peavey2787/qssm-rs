# Proof History

Navigation: [EasyCrypt README](../README.md)

This file preserves the long-form audit log, closure notes, and historical May 2026 research narrative that previously lived in the root README. It is intentionally verbose. The goal of the split is to keep that detail available without forcing every new reader through it before they can run the checker.

Chronology note: this file is intentionally historical. Entries that mention three named axioms, abstract budget nonnegativity assumptions, or half-budget LE arithmetic describe intermediate May 2026 checkpoints and are superseded by the later zero-axiom closure and semantic-budget theorem export recorded further down this file and in the current status docs.
